What is HeinOnline’s Privacy Policy?


We use cookies for authentication purposes.  We also track user activity at the account level; this is done strictly for account usage statistics, which are provided at the request of the account holder.  If you need any further information on this subject, please contact us via email at holsupport@wshein.com.
