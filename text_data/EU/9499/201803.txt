Δέσμευση για την προστασία της ιδιωτικής ζωής 

Η προστασία της ιδιωτικής ζωής σας, είναι σημαντικό για την Betomall Λτδ («η Εταιρεία»). Έχουμε κοινοποιήσει στον Επίτροπο Προστασίας Δεδομένων σχετικά με την επεξεργασία των δεδομένων σύμφωνα με τις απαιτήσεις του Νόμου περί Προστασίας Δεδομένων (Κεφάλαιο 440 της νομοθεσίας της Μάλτας).
Έχουμε προσπαθήσει να κρατήσει τα προσωπικά σας στοιχεία προστατευμένα ανά πάσα στιγμή, με την εφαρμογή κατάλληλων τεχνικών και οργανωτικών ελέγχων.
Αφιερώστε λίγο χρόνο για να διαβάσετε την ακόλουθη πολιτική για να μάθετε πώς χειριζόμαστε τα προσωπικά σας στοιχεία, διότι η χρήση των υπηρεσιών μας υπονοεί την αποδοχή του περιεχομένου της.
Συλλέγουμε και χρησιμοποιούμε τις προσωπικές σας πληροφορίες να ανταποκριθούμε στις νομικές απαιτήσεις, για ιστορικούς και στατιστικούς σκοπούς, για την ασφάλεια και τον έλεγχο, για να σας παρέχουμε τις υπηρεσίες μας, και, για να κάνουμε περισσότερες από αυτές τις υπηρεσίες στη διάθεσή σας στο μέλλον. Από καιρό σε καιρό, μπορούμε επίσης να χρησιμοποιήσουμε τα προσωπικά σας στοιχεία για να επικοινωνήσουμε μαζί σας μέσω τηλεφώνου, ηλεκτρονικού ταχυδρομείου για να σας παρουσιάσουμε τα προϊόντα μας ή για όποια γεγονότα, δραστηριότητες, έργα, σχέδια, τις εξελίξεις, τις επιχειρήσεις και τις ειδικές προσφορές που προωθούνται ή παρέχονται από την Εταιρεία.
Με την παροχή των στοιχείων σας, βεβαιώνετε ότι δεν θεωρείτε τη χρήση των πληροφοριών σας σύμφωνα με την παρούσα Πολιτική Προστασίας Προσωπικών Δεδομένων να αποτελεί παραβίαση των δικαιωμάτων σας σύμφωνα με το Νόμο περί Προστασίας Δεδομένων (Κεφάλαιο 440 της νομοθεσίας της Μάλτας).
Σας ενθαρρύνουμε να επικοινωνήσετε μαζί μας ανά πάσα στιγμή και να σας υπενθυμίσουμε για το δικαίωμά σας να αποχωρήσετε ανά πάσα στιγμή από τη λήψη οποιουδήποτε διαφημιστικού υλικού ή υλικού μάρκετινγκ από εμάς.
Ως πολιτική η Εταιρεία δεν θα αποκαλύψει οποιεσδήποτε προσωπικές πληροφορίες σε οποιονδήποτε άλλον εκτός από εκείνες των εργαζομένων που πρέπει να έχουν πρόσβαση στα δεδομένα σας για να σας παρέχουν υπηρεσίες.
Μπορεί, ωστόσο, να απαιτείται από το νόμο ή νομική διαδικασία για να αποκαλύψουμε τα προσωπικά σας στοιχεία σε αρχές όπως η τοπική Lotteries and Gaming Authority.
Η Εταιρεία εγγυάται την ασφάλεια των δεδομένων που μας στέλνετε με τις φυσικές, ηλεκτρονικές και διαχειριστικές διαδικασίες. Για τη διαβίβαση των ευαίσθητων δεδομένων κρυπτογράφηση SSL χρησιμοποιείται στην ιστοσελίδα μας και να επικοινωνεί με τον επεξεργαστή πληρωμών μας. Παρακαλώ σημειώστε ότι δεν μπορούμε να εγγυηθούμε ότι όλα τα δεδομένα που μεταδίδονται μέσω του Διαδικτύου είναι απολύτως ασφαλή. Ως εκ τούτου, σας ενθαρρύνουμε να λάβετε όλα τα προληπτικά μέτρα για την προστασία των προσωπικών δεδομένων σας, όταν είστε στο διαδίκτυο.
Έχετε πάντα το δικαίωμα πρόσβασης στις πληροφορίες που έχουμε για σας. Για να αναθεωρήσετε και να ενημερώσετε τα προσωπικά στοιχεία επικοινωνίας σας, απλά συνδεθείτε στην ιστοσελίδα μας με το Όνομα Χρήστη και τον Κωδικό σας και μέσα από τη σελίδα Ο λογαριασμός μου μπορείτε να αλλάξετε τις πληροφορίες σας. Αν προτιμάτε μπορείτε να επικοινωνήστε με την υποστήριξη πελατών μας χρησιμοποιώντας τα στοιχεία επικοινωνίας που είναι διαθέσιμα στην ιστοσελίδα. Επιπλέον, έχετε το δικαίωμα να διορθοθούν οποιεσδήποτε ανακριβείς πληροφορίες ή και, κατά περίπτωση, να διαγραφύν.
Είναι δικαίωμά μας να σας ζητήσουμε να μας δώσετε μια γραπτή αίτηση για πληροφορίες που έχουμε για σας.
Η ιστοσελίδα μας χρησιμοποιεί cookies για την παρακολούθηση των επισκεπτών στην ιστοσελίδα μας. Κάθε επισκέπτης στην ιστοσελίδα Betomall θα λαμβάνει ένα cookie στον υπολογιστή του. Αυτό είναι ένα αβλαβές πακέτο που χρησιμοποιείται για αναλυτικούς και στατιστικούς σκοπούς μόνο.
Ένα cookie σύνδεσης χρησιμοποιείται όταν συνδεθείτε στο σύστημα Betomall για να είναι σε θέση να σας αναγνωρίζει όταν χρησιμοποιείτε της ιστοσελίδας. Όταν η σύνδεση λήξε ιτο cookie έχει αφαιρεθεί από τον υπολογιστή σας.
Για να βεβαιωθείτε ότι τα προσωπικά σας στοιχεία παραμένουν εμπιστευτικά μεταφέρουμε αυτές τις κατευθυντήριες γραμμές προστασίας της ιδιωτικής ζωής σε κάθε εργαζόμενο της εταιρείας.
Ο δικτυακός τόπος της Εταιρείας μπορεί, από καιρού εις καιρόν, να περιέχει συνδέσμους προς άλλους δικτυακούς τόπους. Η Εταιρεία δεν μοιράζεται τα προσωπικά στοιχεία σας με αυτές τις ιστοσελίδες και δεν είναι υπεύθυνη για τις πρακτικές προστασίας της προσωπικών δεδομένων από αυτές. Σας ενθαρρύνουμε να μάθετε για τις πολιτικές απορρήτου των εν λόγω εταιρειών.
Αν θέλουμε να χρησιμοποιήσουμε τα προσωπικά σας στοιχεία με διαφορετικό τρόπο από αυτόν που ορίζεται εκείνη τη στιγμή της συλλογής, θα σας ενημερώσουμε σχετικά.
Η Πολιτική Προστασίας Προσωπικών Δεδομένων της Εταιρείας μπορεί να αλλάξει ανά πάσα στιγμή. Είναι προς το συμφέρον σας να αναθεωρήτε την πολιτική προστασίας προσωπικών δεδομένων τακτικά για τυχόν αλλαγές.
