Athan oleh IslamicFinder
Athan ialah aplikasi unggulan dari IslamicFinder. Kami di Athan sangat menghargai privasi pengguna dan mengumpulkan informasi tersebut hanya setelah mereka memberi izi, sehingga mereka dapat menggunakan fitur aplikasi kami dengan maksimal dan mendapatkan pengalaman secara keseluruhan dengan lebih baik. Dimanapun kami mengumpulkan informasi sensitif (seperti lokasi), hal ini dienskripsi dan ditransmisikan kepada kami dengan aman. Kami memerlukan izin untuk mengakses lokasi pengguna, media, kamera, informasi telepon, dan buku alamat. Kami juga memerlukan izin untuk mengirimkan notifikasi dan untuk pembelian fitur tambahan.
Informasi lokasi diperlukan untuk memberikan Jadwal Sholat yang akurat, arah kiblat, informasi mengenai tempat disekitar mereka, dan Kalender Islami untuk penanggalan. Izin notifikasi diperlukan untuk mengirimkan jadwal sholat dan notifikasi pengingat lainnya yang sangat membantu. Akses ke penyimpanan media pengguna diperlukan untuk mengunduh terjemahan Quran dan untuk berbagi jadwal sholat dan doa. Izin kamera pengguna diperlukan untuk memperbaharui gambar profil dan izin pembelian fitur tambahan diperlukan agar pengguna dapat membeli promosi menarik kami. Akses ke informasi telepon dan buku alamat mereka diperlukan untuk membantu mereka menghubungi organisasi islam dan mengundang teman dan keluarga untuk menggunakan aplikasi kami.
Harap dicatat bahwa Athan hanya bisa mengakses apa yang penggunanya inginkan untuk dibagi. Tidak ada informasi yang dilacak atau dibagikan dengan pihak ketiga dan perusahaan lain. Menjaga informasi pribadi pengguna kami tetap aman adalah prioritas kami.

						Penggunaan Cookie
					

						Situs web IslamicFinder menggunakan cookie untuk meningkatkan pengalaman pengguna saat browsing/mengunjungi situs web. Situs web ini menggunakan sistem kontrol cookie yang memungkinkan pengguna untuk membolehkan atau melarang penggunaan cookie pada komputer / perangkat mereka pada kunjungan pertama. Ini mematuhi persyaratan undang-undang terbaru situs web untuk mencapai persetujuan eksplisit dari pengguna mereka sebelum meninggalkan atau membaca file seperti cookies pada komputer / perangkat pengguna.
					

						Cookie adalah file kecil yang disimpan di hard drive komputer pengguna yang melacak dan menyimpan informasi tentang interaksi dan penggunaan situs web pengguna. Hal ini memungkinkan situs web untuk menyediakan penggunanye dengan pengalaman yang disesuaikan.
					

						Agar para pengguna kami dapat melarang atau menghentikan penggunaan dan menyimpan cookie dari situs ini pada hard drive komputer mereka, langkah-langkah yang diperlukan harus diambil pada pengaturan keamanan browser web yang akan memblokir semua cookie situs web dan vendor eksternalnya.
					

						Situs web ini menggunakan software pelacakan untuk memantau pengunjungnya agar dapat lebih memahami penggunaan situs web mereka. Software pelacakan ini disediakan oleh Google Analytics, yang menggunakan cookie untuk memonitor penggunaan pengunjung. Perangkat lunak secara otomatis akan menyimpan cookie ke hard drive komputermu untuk melacak keterlibatan dan penggunaan situs IslamicFinder-mu, tapi itu tidak akan mengumpulkan atau menyimpan informasi pribadi pengguna. Anda dapat membaca kebijakan privasi Google untuk informasi lebih lanjut [http://www.google.com/privacy.html].
					

						Cookie lainnya mungkin disimpan ke hard drive komputermu oleh situs web atau vendor eksternal ketika website ini menggunakan program arahan, tautan atau iklan yang disponsori. Cookie digunakan untuk konversi dan arahan pelacakan, yang biasanya berakhir dalam 30 hari, meskipun beberapa mungkin memakan waktu lebih lama. Tidak ada informasi pribadi dikumpulkan atau disimpan.
